# Student Management System Using FastAPI

## Project Description

Student Management System is a Python-based application developed using FastAPI and CSV file storage.

The system provides both:
- Terminal-based menu system
- REST API using FastAPI

## Features

### Terminal Menu
- Add Student
- View Students
- Search Student
- Update Student
- Delete Student
- Generate Fake Student Data

### FastAPI APIs
- Get all students
- Get student by ID
- Add new student
- Update student
- Delete student
- Generate fake students

## Technologies Used

- Python
- FastAPI
- Uvicorn
- CSV Database
- Faker Library
- Pydantic
